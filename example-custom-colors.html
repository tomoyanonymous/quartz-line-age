Custom Colors Demo

The line age visualization now supports custom colors! You can configure both the fresh (newest) and old (stale) colors using RGB values.

Default: Green → Gray

// Fresh (0 days)
// 90 days old
// 240 days old
// 365+ days old
Fresh: rgb(34, 197, 94)
Old: rgb(156, 163, 175)

Blue → Red

// Fresh (0 days)
// 90 days old
// 240 days old
// 365+ days old
Fresh: rgb(59, 130, 246)
Old: rgb(239, 68, 68)

Purple → Yellow

// Fresh (0 days)
// 90 days old
// 240 days old
// 365+ days old
Fresh: rgb(168, 85, 247)
Old: rgb(234, 179, 8)

Dark → Light (Monochrome)

// Fresh (0 days)
// 90 days old
// 240 days old
// 365+ days old
Fresh: rgb(17, 24, 39)
Old: rgb(226, 232, 240)

Usage Example

import { LineAge } from "quartz-line-age"

// Use custom colors
LineAge({
  enabled: true,
  maxAgeDays: 365,
  freshColor: { r: 59, g: 130, b: 246 },   // blue-500
  oldColor: { r: 239, g: 68, b: 68 },      // red-500
})
