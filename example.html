Quartz Line Age Visualization Demo

This demonstrates the line age visualization feature, similar to Obsidian Git's Line Authoring. Each line has a colored bar on the left that indicates when it was last modified: green for recently updated lines, gradually fading to gray over one year. 

function calculateFibonacci(n) {
  if (n <= 1) return n;
  return calculateFibonacci(n - 1) + calculateFibonacci(n - 2);
}

// Updated recently
function main() {
  console.log(calculateFibonacci(10));
}
